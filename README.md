# SharpSnake 🐍

**SharpSnake** is a modern desktop recreation of the classic arcade game, built entirely from scratch using C# and Windows Presentation Foundation (WPF). Guide your snake, eat the apples, rack up a high score, and avoid crashing into the walls (or yourself)!

## 🌟 Features

* **Classic Arcade Gameplay:** The timeless grid-based movement and growth mechanics you know and love.
* **Native Windows UI:** Built with WPF for a crisp, responsive, and lightweight desktop experience.
* **Score Tracking:** Real-time score updates as your snake grows longer.
* **Keyboard Controls:** Smooth and responsive directional controls using the Arrow Keys or WASD.

## 🚀 Getting Started

### Prerequisites
To run or modify this project, you will need:
* **Windows OS** (WPF is a Windows-only framework)
* **.NET Desktop Runtime** (Version 6.0 or higher recommended)
* **Visual Studio 2022** (If you want to edit the code)

### Installation & Running

#### Option 1: Running the executable
1. Go to the [Releases](../../releases) tab on GitHub.
2. Download the latest `.zip` file.
3. Extract the folder and double-click `SharpSnake.exe` to play!

#### Option 2: Building from source
1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/ManuYT-dev/SharpSnake.git
   ```
2. Open the `.sln` (Solution) file in Visual Studio.
3. Set the build configuration to `Release` or `Debug`.
4. Click **Start** (or press `F5`) to compile and launch the game.
   *(Alternatively, you can run `dotnet run` from the terminal in the project directory).*
